# MEXC P2P Aggregator — Frontend

Vue.js 3 + Quasar Framework 2 + TypeScript 5

## Инициализация

Quasar-проект будет создан в задаче 7.1 через `npm init quasar`.

## Стек

- Vue.js 3 (Composition API)
- Quasar Framework 2
- TypeScript 5
- Orval (API client generation)
- Pinia (state management)
- Vue Query (server state)
