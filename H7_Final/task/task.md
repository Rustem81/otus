# Разработка полнофункционального веб-приложения с использованием AI-агентов

## Цель:
применить весь изученный стек технологий и инструментов для создания полнофункционального приложения;
продемонстрировать навыки работы с AI-агентами на всех этапах разработки;
создать портфолио-проект для демонстрации работодателю.


## Описание/Пошаговая инструкция выполнения домашнего задания:
Проектная работа представляет собой разработку полнофункционального веб-приложения с использованием AI-агентов на всех этапах: от генерации идеи до развертывания в продакшн.


## Вы можете выбрать один из вариантов:

Разработать приложение на основе ДЗ курса - доработать и расширить приложение, созданное в рамках домашних заданий
Создать новое приложение с нуля - выбрать новую идею и реализовать её полностью

## Требования к проекту:

### 1. Функциональность (обязательно)

#### Приложение должно включать:

##### Frontend:
    -Минимум 3 основных экрана/страницы
    -Адаптивный дизайн (десктоп + мобильные устройства)
    -Интерактивные элементы и формы
    -Обработка состояний загрузки и ошибок

##### Backend:
    -База данных с минимум 3 связанными таблицами
    -API с CRUD операциями
    -Аутентификация пользователей
    -Валидация данных

##### Дополнительные функции (минимум 2 из списка):
    -Интеграция с внешними API
    -OAuth2 авторизация
    -Платежная система
    -Аналитика (Яндекс.Метрика, Google Analytics)
    -Файловое хранилище (загрузка изображений/документов)
    -Поиск и фильтрация данных
    -Уведомления (email/push)


### 2. Использование AI-агентов (обязательно)


##### На всех этапах разработки должно быть продемонстрировано использование AI-агентов:

###### Планирование и дизайн:
-Генерация идеи и анализ конкурентов с помощью AI
-Создание UI-концепций через Text-to-UI инструменты
-Создание User Stories и ТЗ с помощью AI

###### Разработка:
-Инициализация проекта с AI-агентом
-Генерация кода компонентов и функций
-Рефакторинг и оптимизация с AI
-Создание тестов с помощью AI

###### Backend и инфраструктура:
-Проектирование БД с AI
-Генерация SQL-схем и миграций
-Создание API endpoints с AI
-Настройка Docker и CI/CD с AI

###### Отладка и оптимизация:
-Использование AI для анализа ошибок
-Оптимизация производительности с AI
-Аудит безопасности с AI


### 3. Технические требования (обязательно)

    Frontend: React, Vue, Next.js или аналогичный современный фреймворк
    Backend: Supabase, Node.js + Express, или другой современный стек
    База данных: PostgreSQL (через Supabase или self-hosted)
    Деплой: Приложение должно быть развернуто и доступно онлайн
    CI/CD: Настроен автоматический деплой
    Версионирование: Код в Git репозитории с понятной историей коммитов

### 4. Документация (обязательно)

##### README.md с описанием проекта:

    -Описание идеи и функциональности
    -Инструкции по установке и запуску
    -Описание технологий
    -Скриншоты интерфейса

##### Документация процесса разработки:

    -Описание использования AI на каждом этапе
    -Примеры промптов и результатов
    -Описание проблем и их решений
    -Выводы и рекомендации


### 5. Качество кода (обязательно)

    Код структурирован и организован
    Используются современные практики разработки
    Код читаемый и прокомментирован где необходимо
    Нет критических багов
    Приложение стабильно работает

## Этапы выполнения проекта:


### Этап 1. Планирование (1 неделя)

    Создание технического задания

    Интеграция с Backend
    Тестирование и отладка

### Этап 4. Разработка Backend (2 недели)

    Развертывание БД
    Создание API
    Настройка безопасности
    Интеграция дополнительных сервисов

### Этап 5. Деплой и финализация (1 неделя)

    Настройка CI/CD
    Развертывание в продакшн
    Финальное тестирование

    Подготовка к защите
